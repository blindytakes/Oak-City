# Oak City Media

Web, video, and photography for Raleigh businesses.

## Tech Stack

- **Next.js 16** (App Router, static export)
- **TypeScript**
- **Tailwind CSS v4**
- **Deployed on Vercel**

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Push to GitHub, connect to [Vercel](https://vercel.com), and it auto-deploys.
